Interesting Features


Probably one the feature I've had the most fun creating and implementing for a client are a series of entertaining character sketches that are drawn overtop of the team members as the users scroll. The drawings were created as SVGs that draw their path with some simple CSS animation. A light touch of JS is used to make sure the animation triggers at the correct point in the page as a user scrolls. Currently the SVG code is directly inline of the HTML; this has the side effect of ballooning the number of code lines in th HTML. While not preferable, with the website itself being only one page, it's fairly easily manageable. In a more in-depth, larger site I would invest more time into injecting the SVG rather than having them inline.

About The Project


A website design for a new animation studio. They're getting a lot of buzz with their new short film and wanted a sleek, modern dozen to draw in traffic and future investors. I designed a professional look and am currently implementing it as a custom WordPress block theme. WordPress' new block themes can provide a lot of flexibility to the client so they're able to make edits and adjustments on their own in the future.

Functionality would need to include a fully fleshed out WP compatible block theme so the client would be able to make many adjustments without coding. Client needs require an ability to add additional pages for future projects, ability to add new products to a WooCommerce store and have them populate to store pages and front page, ability to make adjustments to the home ppage to add current news easily, and not many hard-coded elements, while making sure the theme felt unique.

Design Process


The was fairly straightforward--create a classic looking website that exuded a playful, childlike energy while still having a professional and classic look. After getting some color and shape direction from the clients design team I went through a few mockups. We went back and forth between wildly colorful examples and other dark "modern" looks but eventually settled on a classic white. The client wanted to bring in some color though so we used their branding to give shape and definition to certain sections of the site.
